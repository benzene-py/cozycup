# ☕ Cozy Cup

A modern and responsive coffee shop website built with HTML, CSS, and JavaScript.

Cozy Cup delivers a warm café-inspired experience with an elegant interface, interactive sections, smooth animations, and a fully responsive design across desktop, tablet, and mobile devices.

## 🚀 Live Demo

🔗 https://coozycup.netlify.app

Visit the live website to explore the complete café experience.

## ✨ Features

- Responsive design for desktop, tablet, and mobile devices
- Modern café-themed user interface
- Interactive navigation and smooth scrolling
- Coffee menu showcase
- Reservation section
- Customer testimonials and reviews
- Smooth animations and transitions
- Clean and maintainable code structure
