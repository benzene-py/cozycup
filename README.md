☕ Cozy Cup

Cozy Cup is a modern and responsive coffee shop website built using HTML, CSS, and JavaScript. The project provides an elegant café-themed user experience with interactive menus, table reservations, and a visually appealing design.

✨ Features
  Responsive design for desktop, tablet, and mobile devices
  Interactive coffee menu
  Table reservation interface
  Modern and clean UI
  Smooth animations and transitions
  Organized project structure
  High-quality coffee imagery
  
🛠️ Technologies Used
  HTML5
  CSS3
  JavaScript
